# Booking Payment Client

This bundle is the booking/payment-only frontend.

## Routes
- `/` booking/payment home
- `/submit` public booking/payment submission form
- `/thank-you` separate booking thank you page
- `/admin-login` booking admin login
- `/admin-dashboard` booking dashboard
- `/admin-report` booking report
- `/admin-history` booking history log

## Important
Edit `src/pages/ThankYou.jsx` and replace the developer coffee bank details.

## Local run
```bash
npm install
cp .env.example .env.local
npm run dev
```

## Build
```bash
npm run build
```
